# 🌍 GeoAI

An AI-powered Geospatial Analysis Platform for satellite image analysis using Google Earth Engine.

---

## 🚀 Tech Stack

### Frontend
- React
- TypeScript
- Leaflet

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Python Service
- Flask
- Google Earth Engine

---

## ✨ Features

- User Authentication
- Interactive Map
- Area of Interest (AOI) Selection
- NDVI Analysis
- NDWI Analysis
- Land Use / Land Cover (LULC)
- Report Generation
